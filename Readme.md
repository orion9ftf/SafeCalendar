### Calendario Blindado ###

***Descripción***

Calendario Blindado es una aplicación web desarrollada con Python, Django y PostgreSQL cuyo objetivo es representar matemáticamente el comportamiento de los riesgos cibernéticos asociados a eventos comerciales masivos como Cyberday, Black Friday, Navidad y Día de la Madre.

El sistema permite visualizar eventos comerciales, amenazas asociadas y una representación gráfica basada en funciones cuadráticas que modelan el aumento y disminución del riesgo antes, durante y después de una fecha determinada.

***Objetivo General***

Desarrollar una plataforma web que permita representar matemáticamente el comportamiento de riesgos cibernéticos asociados a eventos comerciales masivos mediante funciones cuadráticas, facilitando la visualización de periodos de mayor exposición a amenazas informáticas.

***Objetivos Específicos***

* Gestionar eventos comerciales relevantes.
* Asociar amenazas de ciberseguridad a cada evento.
* Clasificar eventos según su nivel de riesgo.
* Visualizar gráficamente el comportamiento teórico del riesgo.
* Aplicar conceptos de Cálculo Diferencial mediante funciones cuadráticas.
* Centralizar la administración de eventos y amenazas mediante un panel privado.
